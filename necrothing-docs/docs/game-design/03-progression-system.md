# Progression System

Versione: `v0.1`

## Punti Necro

I Punti Necro rappresentano l'esperienza del custode.

| Azione | XP | Limite |
|---|---:|---|
| Nuovo funerale oggetto fisico | 100 | Nessuno |
| Nuovo funerale oggetto astratto | 50 | 1 al giorno |
| Portare fiori | 10 | 1 volta al giorno per tomba |
| Pulire erbacce | 5 | Solo se presenti |
| Condividere lapide | 25 | 1 al giorno |
| Celebrare anniversario | 30 | 1 volta per anniversario |
| Evento fantasma | 40 | Evento raro |
| Benedizione del prete | 20 | Evento casuale |

## Prestigio del Cimitero

Il prestigio è un punteggio qualitativo separato dagli XP.

Formula indicativa:

```text
prestige =
  graves_count * 2
+ decoration_score
+ cleanliness_score
+ variety_score
+ anniversary_score
+ rare_event_score
```

## Sblocchi

Gli sblocchi devono includere nuove possibilità, non solo cosmetici.

Esempi:

- Rango 2: staccionate, candele, almanacco avanzato.
- Rango 3: becchino, corvi, lapidi gotiche.
- Rango 4: fantasmi, meteo avanzato, mausolei piccoli.
- Rango 5: zone tematiche, mausoleo centrale, eventi rari.
