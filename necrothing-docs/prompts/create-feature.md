# Prompt: Create Feature

Usa esclusivamente la documentazione del repository NECROTHING come fonte di verità.

## Obiettivo

Implementa la feature indicata rispettando:

- architettura feature-first;
- React + TypeScript;
- Zustand per lo stato;
- repository pattern per dati;
- servizi per logica di dominio;
- adapter per differenze web/native;
- test obbligatori.

## Istruzioni

1. Leggi la feature spec in `/specs/features`.
2. Leggi i servizi correlati in `/specs/services`.
3. Leggi il data model in `/specs/data-model`.
4. Genera codice solo per quanto documentato.
5. Non introdurre dipendenze non approvate.
6. Non usare `localStorage`.
7. Non inserire logica di dominio nei componenti React.
8. Scrivi test unitari e, se necessario, test e2e.

## Output atteso

- File creati/modificati.
- Spiegazione breve.
- Test inclusi.
- Eventuali assunzioni esplicite.
